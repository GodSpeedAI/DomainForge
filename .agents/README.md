# .agents

Shared durable state for AI agents working in this repository.

Expected contents:

- `plans/` for implementation plans
- `reports/` for findings, audits, and execution reports
- `specs/` for specs and design notes
- `lessons/` for reusable lessons and recurring pitfalls
- `current_state.md` for checklist-based execution state with evidence
- `next_steps.md` for the next two or three steps and expected outcomes

`AGENTS.md` is the canonical router for how this directory should be used.
